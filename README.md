# A COMPUTER VISION PROJECT

## This repository houses a computer vision project for object detection using Yolo

### Project Overview
This project utilizes the YOLO (You Only Look Once) algorithm for real-time object detection.

### Features
- Real-time object detection
- High accuracy and speed
- Easy integration with various applications

### Installation

1. Clone the repository 
    ```bash
    git clone https://github.com/myingineer/cv_project.git
    cd cv_project
    ```

2. Create a Virtual Environment 
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows use `venv\Scripts\activate`
    ```

3. Install Dependencies 
    ```bash
    pip install -r requirements.txt
    ```
